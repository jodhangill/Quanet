What is Quanet?


Quanet is a codeless interface for applying the NeuroEvolution of Augmenting Topologies (NEAT) algorithm to quantitative trading.

In simpler terms, Quanet is a tool that allows you to use advanced trading algorithms without needing to write any code. It specifically uses a technique called NEAT, which stands for NeuroEvolution of Augmenting Topologies. This technique is used to help improve trading strategies by simulating the process of evolution, allowing the system to learn and adapt over time to find better trading solutions.

In essence, Quanet makes it easy for anyone to harness the power of complex algorithms in trading, even if they don't have programming skills.

How Quanet Works


Initialization

Quanet starts by setting up an initial population of trading strategies, each represented by a neural network. These strategies are random variations, or "genomes," that the algorithm will evaluate and refine over time.

Genome Example

Simulation

The algorithm tests each strategy using the chosen stock data over a specified time range. It simulates trading activities such as buying and selling based on various indicators and tracks how each strategy performs.

Simulated Strategy

Fitness Calculation

After simulating trading, Quanet calculates the overall fitness of each strategy by aggregating performance scores from all the stock data. This score reflects how well the strategy performed across different scenarios.

Evolution

Based on the fitness scores, Quanet evolves the population of strategies. It selects the better-performing strategies to create new "offspring" strategies. These offspring are variations of the best strategies, incorporating mutations and recombinations to explore new possibilities.

Rinse and Repeat

This process repeats for a number of generations. With each generation, Quanet refines and improves the trading strategies by continuously evaluating and evolving them, aiming to find the most effective strategy.

How to Use Quanet


Create Your Fitness Function

Think about what metrics are important for evaluating your trading strategies. Decide on metrics like Sharpe ratio, max drawdown, total returns, and SQN. Input these metrics into Quanet to create your fitness function.

For example, to find a strategy that balances profitability and risk, you could use the fitness function
TOTAL COMPOUND RETURNS/(1 + MAX DRAWDOWN)

Choose Your Stock Data

Select the stocks and time periods you want to analyze. Quanet allows you to pick specific stocks and date ranges for testing your trading strategies.

Ticker Examples

Configure the NEAT Algorithm

Quanet is built on NEAT-Python and provides extensive configuration options for almost all the parameters used in NEAT-Python. To tailor the algorithm to your needs, you can adjust settings such as the maximum number of generations, the fitness threshold, and the population size. For detailed information about these parameters, please see the NEAT-Python documentation.

Run the Algorithm

Once everything is configured, start the algorithm. Quanet will automatically run the simulations, evaluate each trading strategy, and provide you with updates on its progress.
